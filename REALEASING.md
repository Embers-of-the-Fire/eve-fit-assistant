# Releasing

The old version, stored in the `main` branch, is released as three version: `serenity`, `tranquility` and `singularity`.
Their code are stored in the `release-foo` branches.

Each version has a special identifier name and app label.

To release a new version, remember to execute `data/codegen.ps1` to generate new version info, and bump the version in `pubspec.yaml`
